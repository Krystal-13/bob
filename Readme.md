# Spring Boot - Bob Application

레시피를 스크랩하고, 내가 만든 레시피를 공유하는 앱

### 사용 기술
* Spring boot
* Spring Data JPA
* Spring Security
* Jsoup
* MySql

### 프로젝트 기능
* 회원가입 / 로그인 : 비회원은 레시피 검색만 가능
* 공유 레시피 검색
* 레시피 CRUD
* 레시피 북마크
* 리뷰 작성
